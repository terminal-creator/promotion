01

新手村 · 工程素养与 Web 基础

WEEK 1 – 2 · ⏱ 25h · HP+ 基础工程
「先学会把代码跑起来、传上去、给别人看到，再谈架构。」
核心知识点
  • Linux 与命令行：文件/进程/权限/管道/SSHvim · tmux · curl · jq
  • Git 工作流：分支策略、PR、Conflict、Rebase；CI/CD 入门 GitHub Actions
  • 网络基础：TCP 三次握手、HTTP/1.1 vs 2 vs 3、DNS、TLS、CORS
  • FastAPI 起步：路由、依赖注入、Pydantic 模型、错误处理、OpenAPI 文档
  • Python 工程化：venv / poetry、类型注解、ruff + black、pytest
  • API 设计基础：RESTful、状态码、幂等、版本化、分页
小测验

三次握手为什么要 3 次而不是 2 次？

FastAPI 的依赖注入和 Flask 的装饰器路由有什么本质区别？

Git rebase 和 merge 各自的代价是什么？

小项目
P1.1用户系统 API

FastAPI 实现注册/登录/获取信息接口，写 pytest，跑 GitHub Actions CI，自动部署到一台云主机。

验收标准
  • 能用一句 curl 完成注册→登录→拿 token→访问受保护接口
  • 仓库有 README、有测试、CI 绿勾、能复现部署
  • 能讲清楚一个 HTTP 请求从输入域名到返回 JSON 的全过程
